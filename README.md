# Code2Music

Code2Music is a simple frontend-based website where users can play audio and video files directly in the browser.
The goal of the project is to explore layout design, media elements and UI practice using basic web technologies.
There is no backend included.

## Features

- Clean and responsive layout
- Built using HTML, CSS and Bootstrap
- Audio player for music playback
- Video player with browser controls
- Smooth UI components powered by Bootstrap
- Works on mobile and desktop

## Tech Stack
- HTML
- CSS
- Bootstrap 5

## How to Run

Download or clone the project:

```bash
git clone https://github.com/JVKE-KING/Code2Music.git
```

Open index.html in your browser.

## Notes
- This is a static project with no backend or database.
- Media streaming is handled directly by the browser.
- Good for practicing layouts, media elements and frontend structure.
